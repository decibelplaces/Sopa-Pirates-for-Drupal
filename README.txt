This module is an input filter that will change posts containing "SOPA" or "PIPA" on your Drupal site to "Pirate talk"

DEMO: http://netsperience.org/content/blog/test-sopa-pirates-drupal-module

The purpose is to show that your site's friend is the enemy of SOPA and PIPA: PIRATES!

forked by decibel.places from the Talk Like a Pirate Day D6 module created by Richard Eriksson: http://drupal.org/project/pirate

This module has not been tested in D7
